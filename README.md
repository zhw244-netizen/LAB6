# LAB6 -    Lab 6 VERSION CONTRAL

## What this program does 
This program asks the user for their name and age,then prints:
- the user's name
- the user's current age
- the user's age in 20 years

## How to run 
1. open 'main.py'
2. Run the program with Python:
   -'python main.py'
3.Enter your name and age when prompted.
